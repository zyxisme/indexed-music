# Indexed Music

一个简洁的离线音乐播放器，支持本地存储和播放管理。

## 功能特性

- 🎵 本地音乐文件上传和管理
- 🔗 从 URL 下载音乐
- ▶️ 播放控制（播放/暂停/上一首/下一首）
- 📊 进度条拖拽跳转
- 🔊 音量调节
- ⌨️ 键盘快捷键支持
- 💾 IndexedDB 本地存储
- 📱 响应式设计（移动端/平板/桌面）

## 技术栈

- 原生 HTML/CSS/JavaScript
- IndexedDB 本地数据库
- HTML5 Audio API
- Spotify 风格 UI 设计

## 使用方法

直接在浏览器中打开 `indexed-music.html` 即可使用。

## 快捷键

- `空格` - 播放/暂停
- `←` - 后退 5 秒
- `→` - 前进 5 秒

## 浏览器要求

需要支持 IndexedDB 的现代浏览器（Chrome、Firefox、Safari、Edge 等）。
